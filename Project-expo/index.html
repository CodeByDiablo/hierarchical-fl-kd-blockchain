Security for Autonomous Underwater Vehicles

Our Hierarchical
Framework

We introduce a novel "Hierarchical Learning and Security Framework" that addresses the critical trade-off between **model performance** and **resource constraints** in AUVs, while ensuring **data privacy** and **integrity** using a three-stage pipeline.

Key Architectural
Components

Federated Learning (FL)

Collaborative, on-device training by AUV swarms without raw data sharing, preserving mission privacy.

Blockchain Verification

Uses SHA-256 hashing on model updates and an immutable ledger for tamper-free aggregation.

Knowledge Distillation (KD)

Transfers knowledge from a high-capacity teacher model to the lightweight AUV student model for high accuracy.
